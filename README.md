* **Cinematic Dark Mode:** Uses a `#050505` deep black background with high-contrast typography (Clash Display & Satoshi).

* **Fluid Motion:** Inertia-based smooth scrolling to give the site a "luxury" feel.

## üõ† Tech Stack & Libraries
* **Core:** HTML5, CSS3 (CSS Variables, Flexbox, Sticky Positioning), Vanilla JS.
* **GSAP (GreenSock):** Handles complex text reveals and the horizontal "film strip" scroll animation.
* **ScrollTrigger:** Powers the scroll-based animation triggers.
* **Lenis:** Provides the smooth, momentum-based scrolling effect.
* **FontShare API:** Integration for 'Clash Display' and 'Satoshi' fonts.

1.  **Sticky Timeline:** A split-screen experience section where the heading sticks while content scrolls.
